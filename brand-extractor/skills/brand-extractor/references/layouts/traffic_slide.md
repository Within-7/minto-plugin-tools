# 流量分布页布局参考

## 识别关键词
- `流量来源`
- `流量分布`
- `独立站`
- `网站基础数据`
- `日活`
- `跳出率`

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ [品牌简介] [产品定位] [用户定位] [流量模型] [基石流量] [营销活动]    │
├─────────────────────────────────────────────────────────────────────┤
│  独立站流量来源分布                    │  网站基础数据              │
├───────────────────────────────────────┼────────────────────────────┤
│                                       │                            │
│  ┌─────────────────────────────┐     │  日均活跃用户: XXX          │
│  │                             │     │  平均停留时间: XX:XX        │
│  │  流量来源分布柱状图          │     │  人均访问页面: X.XX         │
│  │  (Direct/Search/Social等)   │     │  用户跳出率: XX.XX%         │
│  │                             │     │                            │
│  │  注意：这是柱状图，不是饼图   │     │                            │
│  │                             │     │                            │
│  └─────────────────────────────┘     │                            │
│                                       │                            │
└───────────────────────────────────────┴────────────────────────────┘
```

## 需要提取的内容

| 类型 | 内容 | 存入 |
|------|------|------|
| 图片 | 流量来源分布图 | PNG（**柱状图**） |
| 文本 | 日均活跃用户 | Markdown表格 |
| 文本 | 平均停留时间 | Markdown表格 |
| 文本 | 人均访问页面 | Markdown表格 |
| 文本 | 用户跳出率 | Markdown表格 |

## 重要提示

### ⚠️ 关键区别

| 模式 | 图片 | 图类型 | 内容 |
|------|------|--------|------|
| **品牌详情** | 独立站流量分布 | **柱状图** | Direct/Search/Social等流量来源 |
| **品牌分析** | 外链来源分布 | **饼图** | 外链网站类型分布 |

### 不要搞混！
- 品牌详情的流量图 = **柱状图**（独立站流量来源）
- 品牌分析的外链图 = **饼图**（外链来源分布）

```markdown

|-----|-----|
| 日均活跃用户 | 1,313,333 |
| 平均停留时长 | 04:03 |
| 人均访问页面 | 6.08 |
| 用户跳出率 | 36.69% |
```
