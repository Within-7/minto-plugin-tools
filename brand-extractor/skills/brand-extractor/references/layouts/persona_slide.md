# 用户画像页面布局参考

## 识别关键词
- `用户画像`
- `核心用户画像`
- `Instagram`
- 社交媒体截图

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ [品牌简介] [产品定位] [用户定位] [流量模型] [基石流量] [营销活动]    │
├─────────────────────────────────────────────────────────────────────┤
│  核心用户画像                                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌──────────────────────┐    ┌──────────────────────┐             │
│  │                      │    │                      │             │
│  │  用户画像1            │    │  用户画像2            │             │
│  │  (Instagram截图)      │    │  (Instagram截图)      │             │
│  │                      │    │                      │             │
│  │  显示典型用户的        │    │  显示另一个典型用户    │             │
│  │  社交媒体账号          │    │  的社交媒体账号        │             │
│  │                      │    │                      │             │
│  └──────────────────────┘    └──────────────────────┘             │
│                                                                     │
│  可能附带用户需求文字说明                                            │
└─────────────────────────────────────────────────────────────────────┘
```

## 需要提取的内容

| 类型 | 内容 | 存入 |
|------|------|------|
| 图片 | 用户画像1 | PNG |
| 图片 | 用户画像2 | PNG |
| 文本 | 用户需求说明 | JSON |

## 重要提示
- **需要2张用户画像图片**
- 通常是Instagram或其他社交媒体的截图
- 显示典型用户的社交媒体账号信息
