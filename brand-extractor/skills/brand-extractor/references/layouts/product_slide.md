# 产品展示页布局参考

## 识别关键词
- `产品定位`
- `产品展示`
- `爆款`
- `产品特点`
- `价格`

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ [品牌简介] [产品定位] [用户定位] [流量模型] [基石流量] [营销活动]    │
├─────────────────────────────────────────────────────────────────────┤
│  产品定位 / 产品展示                                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌────────────────────────────────────┐                            │
│  │                                    │   产品名称：XXX            │
│  │                                    │   产品价格：XXX            │
│  │        产品展示图                   │   产品特点：              │
│  │        (大图，通常几MB)             │   1. XXX                  │
│  │                                    │   2. XXX                  │
│  │                                    │   3. XXX                  │
│  │                                    │   ...                     │
│  └────────────────────────────────────┘   7. XXX                   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 需要提取的内容

| 类型 | 内容 | 存入 |
|------|------|------|
| 图片 | 产品展示图 | PNG |
| 文本 | 产品名称 | Markdown |
| 文本 | 产品价格 | Markdown |
| 文本 | 产品特点（7条） | Markdown |

## 重要提示
- 产品图通常是**大尺寸图片**（几MB）
- 产品特点通常有7条
- 注意选择**产品展示图**，不是品牌截图或网页截图
- 对于服务平台（如Chewy订阅服务），产品图可能是服务介绍页面截图
