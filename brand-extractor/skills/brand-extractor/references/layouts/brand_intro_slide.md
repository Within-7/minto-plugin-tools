# 品牌简介页布局参考

## 识别关键词
- `品牌简介`
- `品牌介绍`
- `创立`
- `使命`
- `竞争优势`

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ [品牌简介] [产品定位] [用户定位] [流量模型] [基石流量] [营销活动]    │
├─────────────────────────────────────────────────────────────────────┤
│  品牌简介 / 品牌介绍                                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌──────────┐                                                      │
│  │          │   基本信息：创立于XXXX年，XXX地点，销售额XXX          │
│  │  Logo    │   用户画像：XXX、XXX、XXX                            │
│  │  (小图)   │   品牌使命：XXX                                      │
│  │          │   竞争优势：XXX、XXX、XXX                            │
│  └──────────┘                                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 需要提取的内容

| 类型 | 内容 | 存入 |
|------|------|------|
| 图片 | 品牌Logo | PNG |
| 文本 | 基本信息（创立时间、地点、销售额、市值） | Markdown |
| 文本 | 用户画像描述 | Markdown |
| 文本 | 品牌使命 | Markdown |
| 文本 | 竞争优势 | Markdown |

## 重要提示
- Logo通常是**小尺寸图片**（几KB到几十KB）
- 品牌简介通常在PPT的开头部分（第1-5页）
- 注意区分Logo和产品图，Logo更小
