# 用户定位页面布局参考

## 识别关键词
- `用户定位`
- `势能用户`
- `核心用户`
- `用户画像`

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ [品牌简介] [产品定位] [用户定位] [流量模型] [基石流量] [营销活动]    │
├─────────────────────────────────────────────────────────────────────┤
│  用户定位                                                           │
├────────────────────────────────────┬────────────────────────────────┤
│                                    │                                │
│  ┌────────────────────────────┐   │   用户说明文字                 │
│  │                            │   │   - 势能用户：XXX              │
│  │  用户定位饼图               │   │   - 核心用户：XXX              │
│  │  (用户群体分布)             │   │   - 用户需求：XXX              │
│  │                            │   │                                │
│  │  显示各类用户占比            │   │                                │
│  │                            │   │                                │
│  └────────────────────────────┘   │                                │
│                                    │                                │
├────────────────────────────────────┴────────────────────────────────┤
│  注意：不要提取"用户宠物定位"饼图，只提取"用户定位"饼图           │
└─────────────────────────────────────────────────────────────────────┘
```

## 需要提取的内容

| 类型 | 内容 | 存入 |
|------|------|------|
| 图片 | 用户定位饼图 | PNG |
| 文本 | 用户需求说明（可选） | JSON |

## 重要提示
- **只提取用户定位饼图**，不要提取宠物定位饼图
- 用户定位饼图显示的是用户类型（如：动物救助人士、猫主人、狗主人等）
- 宠物定位饼图显示的是宠物类型（如：猫、狗等）- 不需要
