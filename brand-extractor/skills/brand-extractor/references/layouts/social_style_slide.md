# 社媒风格页面布局参考

## 识别关键词
- 社媒风格
- 内容风格
- 发帖风格
- 内容类型

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ [品牌简介] [产品定位] [用户定位] [流量模型] [基石流量] [营销活动]    │  ← 顶部导航
├─────────────────────────────────────────────────────────────────────┤
│ 任小姐 出海战略咨询                                    Within-7.com  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  社媒风格                                                           │
│                                                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐                 │
│  │             │  │             │  │             │                 │
│  │  [截图1]    │  │  [截图2]    │  │  [截图3]    │                 │
│  │             │  │             │  │             │                 │
│  │  风格标题1  │  │  风格标题2  │  │  风格标题3  │                 │
│  │             │  │             │  │             │                 │
│  └─────────────┘  └─────────────┘  └─────────────┘                 │
│                                                                     │
│  示例标题：                                                          │
│  - 宠物知识介绍                                                      │
│  - 结合可爱宠物图片宣传品牌                                          │
│  - 宠物福祉宣传                                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 需要提取的内容

| 序号 | 图片文件 | 标题字段 |
|------|----------|----------|
| 1 | `{brand}_style_1.png` | `socialStyle.items[0].title` |
| 2 | `{brand}_style_2.png` | `socialStyle.items[1].title` |
| 3 | `{brand}_style_3.png` | `socialStyle.items[2].title` |

## JSON数据结构

```json
{
  "socialStyle": {
    "items": [
      {"title": "宠物知识介绍", "imagePath": "../assets/brand_social/{brand}_style_1.png"},
      {"title": "结合可爱宠物图片宣传品牌", "imagePath": "../assets/brand_social/{brand}_style_2.png"},
      {"title": "宠物福祉宣传", "imagePath": "../assets/brand_social/{brand}_style_3.png"}
    ]
  }
}
```

## 注意事项

1. 通常是3张并排的社媒截图
2. 每张截图下方有风格标题
3. 截图可能是Instagram/TikTok等平台的实际发帖截图
4. 按从左到右顺序提取为 style_1、style_2、style_3
