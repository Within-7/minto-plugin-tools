# SEO页面布局参考

## 识别关键词
- `SEO`
- `关键词`
- `冷流量`
- `Traffic`

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ [品牌简介] [产品定位] [用户定位] [流量模型] [基石流量] [营销活动]    │
├─────────────────────────────────────────────────────────────────────┤
│  流量模型                                                           │
│  1. SEO（冷流量）                                                   │
├─────────────────────────────────────────────────────────────────────┤
│  文字说明：SEO关键词主要集中在XXX等内容...                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌─────────────────────────────────┐  ┌─────────────────────────┐  │
│  │                                 │  │                         │  │
│  │  SEO关键词表格图片               │  │  饼图                   │  │
│  │  (排名/关键词/类型/Traffic)      │  │  非品牌词占比            │  │
│  │  通常是横向表格截图               │  │  (可选)                 │  │
│  │                                 │  │                         │  │
│  └─────────────────────────────────┘  └─────────────────────────┘  │
└─────────────────────────────────────────────────────────────────────┘
```

## 需要提取的内容

| 类型 | 内容 | 存入 |
|------|------|------|
| 文本 | SEO说明文字 | JSON |
| 表格数据 | 关键词、类型、Traffic | JSON |
| 图片 | SEO表格截图 | PNG（可选） |

## 重要提示
- SEO表格**通常是图片格式**，可以直接提取PNG
- 同时需要提取表格中的文字数据存入JSON
- 表格列通常包含：排名、关键词、类型（中文翻译）、Traffic（搜索量）
