# 流量模型页面布局参考

## 识别关键词
- `流量模型`
- `基石流量`
- `品牌宣传`

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ [品牌简介] [产品定位] [用户定位] [流量模型] [基石流量] [营销活动]    │
├─────────────────────────────────────────────────────────────────────┤
│  流量模型                                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  文字说明：                                                         │
│  流量模型由基石流量模型 + 品牌宣传策略组成                          │
│  - 基石流量：XXX                                                    │
│  - 品牌宣传策略：XXX                                                │
│                                                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                               │   │
│  │  流量模型流程图                                                │   │
│  │  (通常是横向流程图)                                            │   │
│  │                                                               │   │
│  │  [SEO] → [外链] → [购买转化] → [复购]                         │   │
│  │                                                               │   │
│  │  显示各流量来源和转化路径                                       │   │
│  │                                                               │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 需要提取的内容

| 类型 | 内容 | 存入 |
|------|------|------|
| 图片 | 流量模型流程图 | PNG |

## 重要提示
- 流量模型图通常是**横向流程图**
- 显示流量来源和转化路径
- 一个PPT通常只有1张流量模型图
