# 外链页面布局参考

## 识别关键词
- `外链`
- `冷流量`
- `合作网站`
- `Top10`

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ [品牌简介] [产品定位] [用户定位] [流量模型] [基石流量] [营销活动]    │
├─────────────────────────────────────────────────────────────────────┤
│  流量模型                                                           │
│  3. 外链（冷流量）                                                  │
├─────────────────────────────────────────────────────────────────────┤
│  外链合作网站的类型：                                               │
│  1. 宠物福利与领养网站                                              │
│  2. 宠物网站与宠物产品电商                                          │
│  3. 维基百科动物相关条目                                            │
│  4. 新闻、杂志、测评类网站                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌──────────────────────┐    ┌──────────────────────────────────┐ │
│  │                      │    │                                  │ │
│  │  外链来源分布饼图     │    │  Top10外链表格                   │ │
│  │  (各类型网站占比)     │    │  (排名/文章题目/网站类型)         │ │
│  │  图片格式 ✓          │    │  注意：可能是PPT原生表格！        │ │
│  │                      │    │  不能直接提取PNG                  │ │
│  └──────────────────────┘    └──────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## 需要提取的内容

| 类型 | 内容 | 存入 |
|------|------|------|
| 图片 | 外链来源分布饼图 | PNG |
| 文本 | 外链合作网站类型说明 | JSON |
| 表格 | Top10外链数据 | JSON（需要截屏才能得到PNG） |

## 重要提示
- **饼图是图片**，可以直接提取
- **Top10表格可能是PPT原生元素**，不是图片！
- 如果表格是原生元素，需要：
  1. 提取表格文字数据存入JSON
  2. 如需PNG图片，需要截屏
