# 封面页布局参考

## 识别关键词
- 封面
- 案例报告
- 品牌名 + 在线平台

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ 任小姐 出海战略咨询                              Within-7.com       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                               │   │
│  │  image1.png (全屏背景图)                                      │   │
│  │  品牌主图/代表性产品图                                        │   │
│  │  大小约2-3MB                                                  │   │
│  │                                                               │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│                     {品牌名} {行业描述}                             │
│                                                                     │
│                     - 案例 报告 {序号} -                            │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 需要提取的内容

| 类型 | 内容 | 存入 |
|------|------|------|
| 图片 | 品牌主图/封面图 | `assets/brands/{brand}.png` |
| 文本 | 品牌英文名 | JSON |
| 文本 | 品牌中文名 | JSON |
| 文本 | 简短介绍（5-10字） | JSON |

## 图片识别

- `image1.png` - 全屏背景图（品牌主图），这是我们要的
- `image2.png` - 左上角logo小图标，不需要

## JSON输出格式

```json
{
  "name": "CHEWY",
  "nameCN": "Chewy 宠物用品在线平台",
  "description": "美国宠物电商龙头"
}
```

## 简短介绍生成规则

从品牌详情中提取核心特点，用5-10个字概括：
- 突出行业地位（龙头/领先）
- 突出业务特点（电商/零售/订阅）
- 突出地区（美国/欧洲/全球）

示例：
- Chewy → 美国宠物电商龙头
- Petco → 全渠道宠物零售
- PetSmart → 全球最大宠物连锁
