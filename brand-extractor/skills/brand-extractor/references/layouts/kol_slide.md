# KOL合作统计页面布局参考

## 识别关键词
- KOL
- 合作
- 社媒
- Instagram / TikTok / YouTube
- 发帖量

## 标准布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│ [品牌简介] [产品定位] [用户定位] [流量模型] [基石流量] [营销活动]    │  ← 顶部导航
├─────────────────────────────────────────────────────────────────────┤
│ 任小姐 出海战略咨询                                    Within-7.com  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  社媒数据                                                           │
│  KOL合作统计                                                        │
│                                                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  摘要文字：                                                          │
│  "品牌在X个社媒渠道的合作数据：合作数量INS最多，发帖量INS最多..."    │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                               │   │
│  │  [柱状图/表格]                                                │   │
│  │  展示各渠道的KOL合作数据                                       │   │
│  │  - 渠道数量                                                   │   │
│  │  - 发帖量                                                     │   │
│  │  - 人均发帖                                                   │   │
│  │                                                               │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 需要提取的内容

| 类型 | 内容 | 存入 |
|------|------|------|
| 图表 | KOL合作统计图 | `assets/brand_social/{brand}_kol_chart.png` |
| 文字 | 数据摘要说明 | JSON `kolStats.summary` |

## 关键词识别

识别"KOL合作"页面时搜索：
- "KOL"
- "合作"
- "社媒"
- "INS" / "Instagram"
- "TikTok"
- "发帖"

## 注意事项

1. 图表可能是柱状图或表格截图
2. 摘要文字通常包含关键统计数据（渠道数、发帖量等）
3. 数据统计通常为最近1年活跃数据
