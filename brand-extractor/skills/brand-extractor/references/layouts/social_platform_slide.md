# 社媒平台数据页面布局参考

## 识别关键词
- 社媒数据
- Instagram / TikTok / YouTube
- 粉丝数
- 发帖频率

┌─────────────────────────────────────────────────────────────────────┐

│ 任小姐 出海战略咨询                                    Within-7.com  │
├─────────────────────────────────────────────────────────────────────┤

## 需要提取的内容（JSON数据）

```json
{
  "socialMediaData": {
    "platforms": [
      {
        "icon": "ig",
        "name": "Instagram",
        "followers": "916,000",
        "posts": "4,738",
        "frequency": "1帖/2天",
        "likes": "14,000"
      },
      {
        "icon": "tt",
        "name": "TikTok",
        "followers": "273,100",
        "posts": "351",
        "frequency": "1帖/3天",
        "views": "3,160,450"
      },
      {
        "icon": "yt",
        "name": "YouTube",
        "followers": "88,400",
        "posts": "1,366",
        "frequency": "1帖/4.3天",
        "views": "53,825"
      }
    ]
  }
}
```

## 数据提取规则

| 平台 | 图标代码 | 必有字段 | 可选字段 |
|------|----------|----------|----------|
| Instagram | `ig` | followers, posts, frequency | likes |
| TikTok | `tt` | followers, posts, frequency | views |
| YouTube | `yt` | followers, posts, frequency | views |

## 注意事项

1. 这页通常没有单独的图片需要提取，数据直接以文字形式记录到JSON
2. 粉丝数、发帖量等数字保持原格式（带逗号、带单位等）
3. 观看量可能使用简写（如 3.16M 表示 3,160,000）
