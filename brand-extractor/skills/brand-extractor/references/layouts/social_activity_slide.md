# 品牌活动页面布局参考

## 识别关键词
- 品牌活动
- 营销活动
- 特色活动
- 活动

│ 任小姐 出海战略咨询                                    Within-7.com  │

│                                                                     │

│                                                                     │

│  │                                                               │   │
│  │  活动标题 1                                                   │   │
│  │  ────────────                                                 │   │
│  │                                                               │   │

│  │                                                               │   │

## 需要提取的内容

每个活动提取：
- 活动标题
- 活动描述
- 活动截图

| 序号 | 图片文件 | JSON字段 |
|------|----------|----------|
| 1 | `{brand}_activity_1.png` | `activity1.imagePath` |
| 2 | `{brand}_activity_2.png` | `activity2.imagePath` |
| 3 | `{brand}_activity_3.png` | `activity3.imagePath` |

## JSON数据结构

```json
{
  "activity1": {
    "title": "活动标题1",
    "description": "活动详细描述...",
    "imagePath": "../assets/brand_social/{brand}_activity_1.png"
  },
  "activity2": {
    "title": "活动标题2",
    "description": "活动详细描述...",
    "imagePath": "../assets/brand_social/{brand}_activity_2.png"
  },
  "activity3": {
    "title": "活动标题3",
    "description": "活动详细描述...",
    "imagePath": "../assets/brand_social/{brand}_activity_3.png"
  }
}
```

## 常见活动类型

- 宠物画像赠送（Pet Portraits）
- 动物福利/公益页面（Gives Back）
- 官方博客/内容营销
- 会员专属活动
- 节日营销活动
- 用户生成内容活动

## 注意事项

1. 通常一个页面展示3个品牌活动
2. 每个活动包含：标题 + 截图 + 描述
3. 截图可能是官网页面、社媒页面或活动海报
4. 按页面从上到下或从左到右顺序提取
