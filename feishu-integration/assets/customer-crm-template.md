## 表格结构

### 主表：客户信息

| 字段名称 | 字段类型 | 必填 | 说明 |
|---------|---------|------|------|
| 客户名称 | 文本 | ✅ | 客户公司或个人名称 |
| 联系人姓名 | 文本 | - | 主要联系人 |
| 联系电话 | 电话 | - | 联系方式 |
| 客户来源 | 单选 | - | 网络推广/客户介绍/展会/主动开发 |
| 客户阶段 | 单选 | - | 潜在客户/意向客户/谈判中/已成交/已流失 |
| 所属行业 | 单选 | - | 互联网/金融/制造/零售/其他 |
| 预估成交金额 | 数字 | - | 单位：元 |
| 负责人 | 文本 | - | 销售负责人 |
| 创建时间 | 日期 | - | 客户建档时间 |
| 备注 | 文本 | - | 其他信息 |

### 关联表：跟进记录

| 字段名称 | 字段类型 | 必填 | 说明 |
|---------|---------|------|------|
| 跟进时间 | 日期 | ✅ | 跟进日期 |
| 跟进方式 | 单选 | - | 电话/微信/邮件/面谈 |
| 跟进内容 | 文本 | ✅ | 跟进详情 |
| 下次跟进时间 | 日期 | - | 计划时间 |
| 关联客户 | 关联 | ✅ | 关联到客户信息表 |

## MCP 工具调用示例

### 创建客户管理系统

```python

# 2. 获取 table_id
get_tables(app_token)

# 3. 添加主表字段

add_table_field(app_token, table_id, "联系人姓名", 1)
add_table_field(app_token, table_id, "联系电话", 11)  # 电话

add_table_field(app_token, table_id, "客户阶段", 3)   # 单选

add_table_field(app_token, table_id, "创建时间", 5)   # 日期

# 4. 添加示例数据
add_record(app_token, table_id, {
```

## 使用建议

1. **定期更新客户阶段**
   - 从"潜在客户" → "意向客户" → "谈判中" → "已成交"
   - 成交后将客户阶段标记为"已成交"

2. **跟进记录管理**
   - 每次跟进后及时记录跟进记录
   - 设置下次跟进时间提醒

3. **数据维护**
   - 及时更新预估成交金额
   - 完善客户行业信息
   - 定期清理"已流失"客户

4. **协作权限**
   - 销售团队：edit 权限
   - 销售经理：full_access 权限
   - 其他部门：view 权限

## 常见场景

### 场景 1: 新增客户

```
1. 调用 add_record() 添加新客户
2. 设置客户阶段为"潜在客户"
3. 记录客户来源和联系方式
4. 分配负责人
```

### 场景 2: 客户跟进

```
1. 调用 get_user_by_email() 获取负责人信息
2. 创建跟进记录关联到客户
3. 更新客户阶段
4. 设置下次跟进时间
```

### 场景 3: 成交转化

```
1. 更新客户阶段为"已成交"
2. 确认实际成交金额
3. 记录成交时间
4. 添加成交备注
```
