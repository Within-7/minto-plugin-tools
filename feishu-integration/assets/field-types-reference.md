## 字段类型列表

| 类型值 | 类型名称 | MCP参数 | 说明 | 使用场景 |
|--------|----------|---------|------|---------|
| 1 | text | field_type=1 | 文本 | 名称、描述、备注 |
| 2 | number | field_type=2 | 数字 | 金额、数量、百分比 |

| 4 | multiSelect | field_type=4 | 多选 | 标签、技能、特性 |

| 15 | progress | field_type=15 | 进度 | 完成度、满意度（0-100%） |

## 字段类型详细说明

### 1. 文本类型 (text, type=1)

**适用场景**：
- 客户名称、项目名称、员工姓名
- 任务描述、备注信息
- 编号、代码等标识

**特点**：
- 支持中英文、数字、符号
- 建议设置字符限制（如 100 字符）

**MCP 调用示例**：
```python
add_table_field(app_token, table_id, "客户名称", 1)
```

---

### 2. 数字类型 (number, type=2)

**适用场景**：
- 采购金额、库存数量
- 工时记录、年龄
- 排序、计算字段

**特点**：
- 支持整数和小数
- 可设置最小值、最大值
- 支持计算公式

**验证规则**：
```python
# 金额字段
min_value = 0  # 最小值
max_value = 1000000  # 最大值

# 数量字段
min_value = 1  # 最小值
```

**MCP 调用示例**：
```python
add_table_field(app_token, table_id, "采购金额", 2)
```

---

### 3. 单选类型 (select, type=3)

**适用场景**：
- 订单状态（待审批/已批准/已完成）
- 客户阶段（潜在客户/意向客户/已成交）
- 优先级（高/中/低）
- 部门、类别等分类信息

**特点**：
- 预设选项列表
- 单选，互斥
- 可设置默认值

**选项配置**：
```python
options = ["选项1", "选项2", "选项3"]
```

**MCP 调用示例**：
```python
add_table_field(app_token, table_id, "订单状态", 3)
# 在飞书界面中配置选项：待审批、已批准、已完成
```

---

### 4. 多选类型 (multiSelect, type=4)

**适用场景**：
- 技能标签（Python/Java/Go）
- 产品特性（性能好/易用/稳定）
- 服务区域（华北/华东/华南）

**特点**：
- 可选择多个选项
- 适用于标签、属性

**MCP 调用示例**：
```python
add_table_field(app_token, table_id, "技能标签", 4)
```

---

### 5. 日期类型 (dateTime, type=5)

**适用场景**：
- 创建时间、更新时间
- 截止日期、生日
- 入职日期、订单日期

**特点**：
- 支持日期+时间
- 可设置默认值（如"今天"）
- 支持时间范围验证

**日期格式**：
```python
# MCP 工具中使用毫秒时间戳
timestamp = 1704067200000  # 2024-01-01 00:00:00
```

**MCP 调用示例**：
```python
add_table_field(app_token, table_id, "创建时间", 5)
```

## 字段验证规则

### 数字字段

```python
# 金额字段
min_value = 0  # 不能为负数
decimal_places = 2  # 保留两位小数

# 数量字段
min_value = 1  # 至少为1
integer_only = True  # 仅整数
```

### 文本字段

```python
# 必填字段
required = True

# 字符限制
max_length = 100  # 最大100字符
```

### 日期字段

```python
# 时间范围验证
# 结束日期不能早于开始日期
end_date >= start_date

# 默认值
default = "today"  # 今天
```

## MCP 工具调用完整示例

```python
# 1. 创建多维表格
app_token = create_bitable(name="客户管理系统")

# 2. 获取 table_id
tables = get_tables(app_token)
table_id = tables[0]["table_id"]

# 3. 添加字段
add_table_field(app_token, table_id, "客户名称", 1)      # 文本
add_table_field(app_token, table_id, "联系电话", 11)    # 电话
add_table_field(app_token, table_id, "客户阶段", 3)     # 单选
add_table_field(app_token, table_id, "预估金额", 2)     # 数字
add_table_field(app_token, table_id, "创建时间", 5)     # 日期

# 4. 添加记录
add_record(app_token, table_id, {
    "客户名称": "示例科技公司",
    "联系电话": "13800138000",
    "客户阶段": "意向客户",
    "预估金额": 50000,
    "创建时间": 1704067200000  # 毫秒时间戳
})
```

## 最佳实践

1. **字段命名规范**
   - 使用清晰的中文描述
   - 避免缩写和特殊字符
   - 保持命名一致性

2. **字段类型选择**
   - 优先使用合适的专用类型
   - 电话用电话类型，邮箱用邮箱类型
   - 状态字段用单选类型

3. **必填字段设置**
   - 关键字段设为必填
   - 如：客户名称、订单号、员工姓名

4. **字段顺序**
   - 重要字段放前面
   - 按使用频率排序
   - 关联信息放后面

5. **验证规则**
   - 数字字段设置最小值
   - 文本字段限制长度
   - 日期字段设置范围
