## 搜索技巧

### site: 限定搜索
```
site:gsxt.gov.cn 公司名称     # 工商信息
site:sse.com.cn 公司简称       # 上市公司公告
site:36kr.com 公司名 融资      # 融资新闻
site:zhihu.com 品牌 评价       # 用户评价
```

### 时间限定
```
公司名 年报 2024..2025         # 限定时间范围
公司名 融资 after:2024/1/1     # 2024年后
```

### 组合搜索
```
"公司全称" 营收 OR 收入 site:sse.com.cn
品牌名 ("用户评价" OR "口碑" OR "体验")
```

## 信息验证原则

1. **官方优先**：政府/交易所信息 > 商业数据库 > 媒体
2. **交叉验证**：至少2个独立来源确认
3. **时效标注**：财务数据必须标注年份
4. **冲突说明**：来源冲突时明确标注并说明
