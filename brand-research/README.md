# Brand Research - 竞品品牌研究插件

Claude Code 插件，通过 AI 网络搜索进行专业的竞品品牌分析。

## 功能特性

- 🔍 **全面调研**：品牌成长历程、融资、财务、用户心智、产品定价
- ✅ **多源验证**：政府官方、交易所、商业数据库、媒体多渠道交叉验证
- 📊 **结构化报告**：生成专业 Markdown 分析报告
- 🏷️ **可信度标注**：每个数据点标注来源和可信等级

## 安装

```bash
# 复制到项目 .claude 目录
cp -r brand-research /path/to/your-project/.claude/plugins/

# 或复制组件
cp commands/*.md /path/to/your-project/.claude/commands/
cp -r skills/brand-research /path/to/your-project/.claude/skills/
```

## 使用方法

```
/brand-analyze <品牌名>
```

示例：
```
/brand-analyze 瑞幸咖啡
/brand-analyze 小米
/brand-analyze 蔚来汽车
```

## 分析维度

| 维度 | 内容 | 信息源 |
|------|------|-------|
| 品牌概况 | 公司信息、Slogan、定位 | 官网、工商系统 |
| 成长历程 | 里程碑、时间线 | 官网、媒体报道 |
| 融资情况 | 轮次、金额、投资方 | IT桔子、36氪、天眼查 |
| 财务数据 | 年营收、增长率 | 财报、财经媒体 |
| 用户心智 | 品牌形象、口碑 | 社交媒体、电商评论 |
| 核心产品 | 产品线、定位 | 官网、电商平台 |
| 定价策略 | 价格区间、促销 | 电商平台、官网 |

## 信息源优先级

1. **政府官方** - 国家企业信用信息公示系统、证监会、知识产权局
2. **交易所公告** - 上市公司年报、招股书
3. **商业数据库** - 天眼查、企查查、IT桔子
4. **财经媒体** - 财新、界面、36氪
5. **社交媒体** - 微博、小红书、知乎（用户心智）

## 输出示例

分析报告将保存到：`docs/brand-analysis/{品牌名}-分析报告-{日期}.md`

报告包含：
- 品牌概况表
- 成长时间线
- 融资历史
- 财务数据（含可信度）
- 用户心智分析
- 产品矩阵
- 定价策略
- SWOT 分析
- 信息来源汇总
