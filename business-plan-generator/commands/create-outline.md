# create-outline - 创建商业计划书大纲

通过交互式询问的方式,创建适用于指定行业的商业计划书大纲框架。

## 用法

```bash
/plugin:business-plan-generator:create-outline
```

执行后会依次询问以下问题:

### 1. 行业类型
```
请问您的业务属于哪个行业?
1. pet - 宠物行业
2. ecommerce - 电商行业
3. saas - SaaS/软件行业
4. restaurant - 餐饮行业
5. education - 教育行业
6. healthcare - 医疗行业
7. retail - 零售行业
8. 其他(请输入行业名称)

(请输入选项编号或行业名称)
```

### 2. 企业类型
```
请问企业类型是什么?
1. startup - 初创企业
2. expansion - 业务扩张
3. investment - 融资计划
4. restructuring - 业务重组

(请输入选项编号,默认为初创企业)
```

### 3. 大纲调整
```
以下是为您生成的商业计划书大纲:

[显示大纲结构]

请问是否需要调整大纲?
- 输入"是"或"Y"进行调整
- 输入"否"或"N"或直接跳过使用当前大纲
```

### 4. 数据补充
```
请问是否有市场数据需要补充?
- 输入"是"或"Y" → 引导使用 /plugin:business-plan-generator:add-data
- 输入"否"或"N" → 引导使用 /plugin:business-plan-generator:generate-plan
```

## 流程

1. 用户执行 `/plugin:business-plan-generator:create-outline` 命令
2. Agent依次询问上述4个问题
3. 用户回答每个问题
4. 根据行业类型和企业类型生成对应大纲
5. 显示大纲结构
6. 询问用户是否需要调整大纲
7. 询问用户是否有数据要补充
8. 根据用户回答引导下一步操作
9. 保存大纲供后续使用

## 输出

### 行业大纲框架

根据行业类型,提供对应的商业计划书大纲:

#### 通用框架(所有行业)
```
1. 执行摘要
2. 公司描述
3. 市场分析
4. 产品/服务
5. 市场推广策略
6. 运营计划
7. 财务计划
8. 风险管理
```

#### 行业特定部分

不同行业可能有额外的章节:

**宠物行业**:
- 宠物市场细分
- 宠物用品vs宠物服务
- 宠物健康趋势

**电商行业**:
- 平台策略(自营/第三方)
- 供应链管理
- 物流体系
- 技术架构

**SaaS行业**:
- 产品功能模块
- 客户成功指标
- 技术架构

**餐饮行业**:
- 菜单设计
- 选址策略
- 供应链管理

## 示例

### 宠物行业初创企业大纲

```bash
用户: /plugin:business-plan-generator:create-outline

系统: 请问您的业务属于哪个行业?
用户: pet

系统: 请问企业类型是什么?
用户: startup

系统: 以下是为您生成的商业计划书大纲:
# 宠物行业商业计划书(初创企业)

## 1. 执行摘要
- 项目概述
- 核心亮点
- 融资需求

## 2. 公司描述
- 业务模式
- 愿景使命
- 法律结构

## 3. 市场分析
- 宠物市场规模
- 目标客户画像
- 市场趋势

## 4. 产品/服务
- 宠物用品
- 宠物服务
- 独特卖点

## 5. 市场推广策略
- 定价策略
- 渠道策略
- 营销计划

## 6. 运营计划
- 团队结构
- 运营流程
- 里程碑

## 7. 财务计划
- 收入预测
- 成本分析
- 盈利模型

## 8. 风险管理
- 市场风险
- 运营风险
- 应对策略

系统: 请问是否需要调整大纲?
用户: 否

系统: 请问是否有市场数据需要补充?
用户: 是

系统: 请使用 /plugin:business-plan-generator:add-data 添加数据
```

## 注意事项

- 大纲是框架,不是固定模板
- 用户可以调整章节顺序和内容
- 后续添加的数据会自动填充到大纲中
- 不同行业的大纲结构会有差异
- 用户可以选择默认选项(初创企业)
