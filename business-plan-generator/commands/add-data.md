# add-data - 添加数据

通过交互式询问的方式,收集商业计划书所需的行业数据,用于后续生成计划书。

## 用法

```bash
/add-data
```

执行后会依次询问以下问题:

### 1. 市场规模
```
请问目标市场的规模是多少?
(例如: 5000万、1000万、200万)
```

### 2. 竞品数量
```
主要竞争对手有多少个?
(例如: 5个、10个)
```

### 3. 市场增长率
```
市场年增长率是多少?
(例如: 15%、20%、25%)
```

### 4. 初始投资
```
计划初始投资多少?
(例如: 200万、500万、1000万)
```

### 5. 定价策略
```
产品/服务的平均定价是多少?
(例如: 150元、500元、2000元)
```

### 6. 毛利率
```
预期毛利率是多少?
(例如: 25%、30%、40%)
```

### 7. 目标客户
```
请描述目标客户画像
(例如: 25-35岁城市白领、月收入8000-15000元)
```

### 8. 自定义数据
```
还有其他需要补充的数据吗?
(可以输入任何相关信息,或直接跳过)
```

## 流程

1. 用户执行 `/add-data` 命令
2. Agent依次询问上述8个问题
3. 用户回答每个问题(或选择跳过)
4. Agent记录所有回答到临时数据存储
5. 确认数据已添加,询问是否需要补充更多数据
6. 用户可以多次执行 `/add-data` 添加数据

## 数据使用

添加的数据会在以下场景使用:

### 市场分析部分
- 市场规模 → 计算市场占有率
- 竞品数量 → 竞品分析
- 增长率 → 市场预测

### 财务计划部分
- 初始投资 → 现金流分析
- 定价 → 收入预测
- 毛利率 → 盈利分析

### 营销策略部分
- 目标客户画像 → 客户获取策略

## 注意事项

- 数据是临时存储,不持久化
- 每次生成计划书都会使用最新数据
- 可以使用 `/plugin:business-plan-generator:generate-plan` 重新生成
- 数据可以多次添加和覆盖
- 用户可以选择跳过任何问题
