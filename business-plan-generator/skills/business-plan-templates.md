# 商业计划书模板库

本技能提供各行业的商业计划书模板和框架,帮助快速生成专业、结构化的商业计划书。

## 行业模板库

## 通用章节模板

### 执行摘要模板
```
[项目名称]是一个[行业]领域的[初创/扩张/投资]项目,致力于[核心价值主张]。本项目计划[融资需求]用于[核心用途]。

**核心亮点**:
• [亮点1]: [详细描述]
• [亮点2]: [详细描述]
• [亮点3]: [详细描述]

**融资需求**:
[融资金额]用于[资金用途分配]
```

### 市场分析模板
```
**市场规模**:
根据[数据来源]的数据,目标市场的总规模为[金额],预计年增长率为[X]%。

**目标客户**:
核心客户群体为[客户画像],特征包括[年龄/性别/地域/收入/行为]。

**市场趋势**:
[行业]市场呈现[趋势1]/[趋势2]等趋势,为[产品/服务]创造了[机会]。
```

## 使用指南

### 如何选择模板

1. **识别行业**: 确定你的业务属于哪个行业
2. **选择对应模板**: 使用该行业的标准大纲
3. **定制内容**: 根据具体业务调整章节
4. **填充数据**: 使用用户提供的真实数据

### 数据填充技巧

- 使用具体数据而非泛泛描述
- 引用数据来源增强可信度
- 使用数据支撑你的分析和建议
- 提供合理的假设和预测

### 最佳实践

- 章节清晰,逻辑严密
- 数据驱动,避免空泛
- 突出竞争优势和差异化
- 提供可行的实施计划
