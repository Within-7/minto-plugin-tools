# 增长趋势图表 - 数据格式说明

## 标准数据结构

```javascript
const chartData = {
  title: "图表标题",
  unit: "单位（如：亿元、万、百万）",
  categories: ["2019", "2020", "2021", "2022", "2023"],
  series: {
    value: {
      name: "金额(亿元)",
      data: [100, 150, 220, 310, 450]
    },
    growth: {
      name: "增速(%)",
      data: [20.5, 50.0, 46.7, 40.9, 45.2]
    }
  }
};
```

---

## 输入格式转换

### 格式一：表格文本

**输入：**
```
年份    金额    增速
2019    100     20.5%
2020    150     50%
2021    220     46.7%
```

**解析规则：**
- 第一列为年份（categories）
- 第二列为金额（series.value.data）
- 第三列为增速（series.growth.data）
- 去除百分号，转为数值

---

### 格式二：列表文本

**输入：**
```
2019年：100亿，增速20.5%
2020年：150亿，增速50%
2021年：220亿，增速46.7%
```

**解析规则：**
- 正则提取年份：`(\d{4})年?`
- 正则提取金额：`[：:]\s*([\d.]+)`
- 正则提取增速：`增速\s*([\d.]+)%?`

---

### 格式三：逗号分隔

**输入：**
```
年份：2019,2020,2021,2022,2023
金额：100,150,220,310,450
增速：20.5,50,46.7,40.9,45.2
```

**解析规则：**
- 按行解析，逗号分隔
- 自动识别标签（年份/金额/增速）

---

### 格式四：纯数字序列

**输入：**
```
金额：100, 150, 220, 310, 450
增速：20.5%, 50%, 46.7%, 40.9%, 45.2%
年份范围：2019-2023
```

**解析规则：**
- 根据年份范围自动生成 categories
- 解析数值和增速数组

---

## 数据清洗规则

### 1. 百分比处理

```javascript
// 输入可能是以下格式
"57.8%"   → 57.8
"0.578"   → 57.8 (自动识别)
"57.8"    → 57.8

// 转换逻辑
function parsePercent(value) {
  if (typeof value === 'string') {
    value = value.replace('%', '');
  }
  const num = parseFloat(value);
  // 如果是小数形式（<1），转为百分比
  if (num < 1 && num > -1) {
    return num * 100;
  }
  return num;
}
```

### 2. 金额处理

```javascript
// 支持的单位
"100万"     → 100 (unit: "万")
"1.5亿"     → 1.5 (unit: "亿")
"578.31百万" → 578.31 (unit: "百万")
"100"       → 100 (unit: "")

// 解析逻辑
function parseValue(value) {
  const match = value.toString().match(/([\d.]+)\s*(万|亿|百万|千万)?/);
  return {
    value: parseFloat(match[1]),
    unit: match[2] || ""
  };
}
```

### 3. 年份处理

```javascript
// 支持的格式
"2019"      → "2019"
"2019年"    → "2019"
"19年"      → "2019"
"2024E"     → "2024E"  // 预测值保留E
"2024E年"   → "2024E"
```

## 数据验证

### 必填项检查

| 字段 | 是否必填 | 验证规则 |
|------|---------|---------|
| categories | ✅ | 长度 >= 2 |
| value.data | ✅ | 与 categories 长度一致 |
| growth.data | ✅ | 与 categories 长度一致 |

### 数值范围检查

| 字段 | 有效范围 | 异常处理 |
|------|---------|---------|
| value.data | > 0 | 警告但允许 |
| growth.data | -100 ~ 500 | 超出范围提示确认 |

### 一致性检查

```javascript
// 检查数据长度一致
if (value.data.length !== categories.length) {
  throw new Error("金额数据与年份数量不一致");
}
if (growth.data.length !== categories.length) {
  throw new Error("增速数据与年份数量不一致");
}
```

## 特殊场景处理

### 1. 缺失数据

```javascript
// 输入包含缺失值
"2020年：-，增速-"

// 处理方式：使用 null，图表自动断开
data: [100, null, 220, 310, 450]
```

### 2. 负增长

```javascript
// 负增速正常显示
growth.data: [20, -15, 30, -5, 10]
```

### 3. 预测数据标记

```javascript
// 预测年份加 E 后缀
categories: ["2019", "2020", "2021", "2022E", "2023E"]

// 可选：预测数据使用虚线
lineStyle: {
  type: index >= firstForecastIndex ? 'dashed' : 'solid'
}
```
