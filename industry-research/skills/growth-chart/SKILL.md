# 行业增长趋势图表生成技能

## 概述

这是一个专门用于生成"行业增长趋势+年度增长率"双Y轴组合图表的技能。用户提供任意行业的数据，自动生成专业的 ECharts 可视化图表。

## 核心能力

### 图表特征

- **图表类型**：柱状图 + 折线图组合
- **双Y轴设计**：左轴显示金额/数值，右轴显示增速百分比
- **数据标签**：自动显示每个数据点的数值
- **渐变配色**：专业商业报告风格

### 适用场景

| 场景 | 示例 |
|------|------|
| 市场规模增长 | 行业市场规模年度变化 |
| 营收增长 | 企业营收及增长率 |
| 用户增长 | 用户数及增长率 |
| 产量增长 | 产量及同比增长率 |
| 交易额增长 | GMV及增速变化 |

## 数据输入格式

支持多种输入格式：

### 格式一：表格形式（推荐）

```
年份    金额    增速
2015    578.31    57.80%
2016    773.13    33.70%
2017    1118.42   44.70%
...
```

### 格式二：列表形式

```
2015年：578.31百万，增速57.80%
2016年：773.13百万，增速33.70%
2017年：1118.42百万，增速44.70%
...
```

### 格式三：JSON 格式

```json
{
  "title": "新能源汽车市场规模",
  "unit": "亿元",
  "data": [
    {"year": "2015", "value": 578.31, "growth": 57.8},
    {"year": "2016", "value": 773.13, "growth": 33.7}
  ]
}
```

### 格式四：分开提供

```
金额数据：578.31, 773.13, 1118.42, ...
增速数据：57.8%, 33.7%, 44.7%, ...
年份：2015-2024
```

## 输出内容

### 1. 完整 HTML 文件

可直接在浏览器打开运行的图表文件。

### 2. 图表特点

- 蓝色渐变柱状图：显示金额/数值数据
- 红色折线图：显示增速百分比
- 双Y轴：左侧金额，右侧百分比
- 数据标签：每个数据点显示数值
- 响应式设计：自适应屏幕宽度

## 可配置项

| 配置项 | 默认值 | 说明 |
|-------|-------|------|
| 图表标题 | "市场规模及增速趋势" | 可自定义 |
| 金额单位 | "百万" | 支持万、亿等 |
| 柱状图颜色 | 蓝色渐变 | 可自定义 |
| 折线图颜色 | 红色 | 可自定义 |
| 图表宽度 | 900px | 自适应 |
| 图表高度 | 500px | 可调整 |

## 处理逻辑

### Step 1: 解析输入

1. 识别年份/时间维度
2. 提取金额/数值数据
3. 提取增速百分比数据
4. 识别单位（万、亿、百万等）

### Step 2: 数据验证

1. 检查数据完整性
2. 验证数值格式
3. 检查增速合理性（-100% ~ 500%）
4. 处理缺失数据

### Step 3: 生成图表

1. 创建双Y轴配置
2. 配置柱状图系列
3. 配置折线图系列
4. 添加数据标签
5. 设置配色方案

### Step 4: 输出文件

1. 生成完整HTML代码
2. 保存到指定目录
3. 提供使用说明

## 注意事项

1. **数据一致性**：确保金额和增速数据年份对应
2. **单位标注**：在Y轴标题中明确单位
3. **增速格式**：输入可以是小数(0.578)或百分比(57.8%)
4. **预测数据**：预测年份建议加"E"后缀（如2024E）
5. **负增长**：支持负增长率显示
