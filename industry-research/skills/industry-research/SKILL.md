# 行业研究技能

## 概述

这是一个专业的行业研究与商业咨询技能，通过 AI 网络搜索和咨询方法论，为用户提供全面、深入的行业分析报告。

## 核心能力

### 1. 行业综合研究
- 行业概况与市场规模
- 发展历程与阶段判断
- 产业链分析
- 政策环境分析

### 2. 全球市场分析
- 全球市场规模与格局
- 重点区域市场分析
- 中国市场地位
- 进出口贸易分析

### 3. 行业热点与趋势
- 政策动态追踪
- 技术创新突破
- 资本市场动向
- 行业趋势预测

### 4. 重点产品分析
- 产品矩阵梳理
- 技术路线对比
- 创新方向分析

### 5. 竞争格局分析
- 市场集中度
- 头部企业研究
- 竞争态势分析
- 波特五力模型

### 6. 消费者洞察
- 用户画像构建
- 消费行为分析
- 需求动机挖掘
- 决策路径绘制

### 7. 品牌案例拆解
- 发展历程复盘
- 商业模式分析
- 关键成功因素
- 出海战略借鉴

## 咨询方法论

### PESTEL 分析框架

```
Political     政策环境 - 监管政策、产业政策
Economic      经济环境 - 宏观经济、消费能力
Social        社会环境 - 人口结构、消费观念
Technological 技术环境 - 技术创新、技术趋势
Environmental 环境因素 - 环保要求、可持续发展
Legal         法律因素 - 行业法规、合规要求
```

### 市场规模分析

```
TAM (Total Addressable Market)     总可触达市场
SAM (Serviceable Addressable Market) 可服务市场
SOM (Serviceable Obtainable Market)  可获得市场
```

### 竞争分析框架

- **波特五力模型**：现有竞争者、潜在进入者、替代品、供应商、买方
- **SWOT 分析**：优势、劣势、机会、威胁
- **竞争对手矩阵**：多维度对比分析

### 消费者研究框架

- **用户画像**：人口统计 + 心理特征 + 行为特征
- **JTBD**：功能性任务 + 情感性任务 + 社会性任务
- **消费者旅程**：认知 → 考虑 → 购买 → 使用 → 忠诚

## 注意事项

1. **数据时效性**：市场数据必须标注年份和来源
2. **交叉验证**：关键数据需多源验证
3. **区分事实与观点**：明确区分客观数据与主观判断
4. **本地化视角**：结合中国市场实际情况分析
5. **出海参考**：关注品牌出海案例与经验借鉴
