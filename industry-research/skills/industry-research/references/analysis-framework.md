# 行业研究分析框架

## 宏观分析框架

### PESTEL 分析

| 维度 | 分析内容 | 关注点 |
|------|---------|-------|
| Political 政策 | 监管政策、产业政策、贸易政策 | 政策支持/限制、合规要求 |
| Economic 经济 | GDP、消费水平、通胀、利率 | 市场购买力、经济周期 |
| Social 社会 | 人口结构、消费观念、生活方式 | 需求变化、消费趋势 |
| Technological 技术 | 技术创新、技术成熟度、专利 | 技术驱动、颠覆风险 |
| Environmental 环境 | 环保要求、可持续发展 | ESG、绿色转型 |
| Legal 法律 | 行业法规、合规要求 | 准入门槛、合规成本 |

### 行业生命周期

```
导入期 → 成长期 → 成熟期 → 衰退期

判断标准：
- 市场增长率
- 竞争者数量
- 产品标准化程度
- 利润率水平
```

## 市场分析框架

### 市场规模估算

**方法一：自上而下**
```
TAM = 总人口 × 目标人群比例 × 客单价 × 购买频次
```

**方法二：自下而上**
```
TAM = ∑(各渠道/区域销售额)
```

**方法三：类比法**
```
参考相似市场/国家的发展路径进行估算
```

### SAM/SOM 估算

```
SAM = TAM × 渠道可触达率 × 区域覆盖范围
SOM = SAM × 预期市场份额
```

## 竞争分析框架

### 波特五力模型

| 力量 | 分析维度 | 强度判断 |
|------|---------|---------|
| 现有竞争者 | 竞争者数量、差异化程度、退出壁垒 | 高/中/低 |
| 潜在进入者 | 进入壁垒、资本需求、政策限制 | 高/中/低 |
| 替代品威胁 | 替代品数量、转换成本、性价比 | 高/中/低 |
| 供应商议价 | 供应商集中度、差异化、转换成本 | 高/中/低 |
| 买方议价 | 买方集中度、价格敏感度、转换成本 | 高/中/低 |

### 竞争对手矩阵

```
           高 ← 市场份额 → 低
        ┌─────────────────────────┐
   高   │     领导者    │     挑战者     │
   ↑    ├─────────────────────────┤
增长率  │     利基者    │     跟随者     │
   低   │              │              │
        └─────────────────────────┘
```

### SWOT 分析

| | 正面 | 负面 |
|--|------|------|
| **内部** | Strengths 优势 | Weaknesses 劣势 |
| **外部** | Opportunities 机会 | Threats 威胁 |

## 消费者分析框架

### 用户画像维度

```
人口统计：年龄、性别、收入、职业、教育、地域
心理特征：价值观、生活方式、个性特征
行为特征：购买频次、客单价、品牌偏好、渠道偏好
需求特征：功能性需求、情感性需求、社会性需求
```

### JTBD (Jobs to Be Done)

| 任务类型 | 问题示例 |
|---------|---------|
| 功能性任务 | 用户想完成什么具体任务？ |
| 情感性任务 | 用户想获得什么感受？ |
| 社会性任务 | 用户想给他人什么印象？ |

### 消费者旅程

```
认知 → 考虑 → 购买 → 使用 → 忠诚

每个阶段关注：
- 触点（在哪里接触信息）
- 行为（做了什么）
- 情绪（什么感受）
- 痛点（什么问题）
```

|------|---------|
