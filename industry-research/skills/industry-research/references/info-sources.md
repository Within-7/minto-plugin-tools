## 搜索技巧

### site: 限定搜索
```
site:stats.gov.cn 行业数据
site:mckinsey.com China retail
site:ir.baidu.com 年报
```

### 时间限定
```
行业研究报告 2024..2025
market size after:2024/1/1
```

### 文件类型
```
filetype:pdf 行业研究报告
filetype:pptx 行业分析
```

### 组合搜索
```
("市场规模" OR "市场总额" OR "TAM") 行业名称
行业名称 ("报告" OR "白皮书" OR "研究")
```

## 数据验证原则

1. **官方优先**：政府数据 > 行业协会 > 研究机构 > 媒体
2. **交叉验证**：至少2个独立来源确认
3. **时效标注**：标注数据年份和来源
4. **方法透明**：了解数据统计口径
5. **谨慎使用**：付费报告的公开摘要数据可能有限
