## CSS @page 配置

### 基础配置

```css
/* A4 纵向 */
@page {
  size: A4 portrait;
  margin: 20mm;
}

/* A4 横向 */
@page {
  size: A4 landscape;
  margin: 20mm;
}

/* A3 纵向 */
@page {
  size: A3 portrait;
  margin: 20mm;
}

/* 自定义尺寸 */
@page {
  size: 210mm 297mm;
  margin: 20mm 15mm;
}
```

### 分页控制

```css
/* 避免元素内分页 */
.no-break {
  page-break-inside: avoid;
  break-inside: avoid;
}

/* 强制在元素前分页 */
.page-break-before {
  page-break-before: always;
  break-before: page;
}

/* 强制在元素后分页 */
.page-break-after {
  page-break-after: always;
  break-after: page;
}
```

## 打印DPI参考

| 用途 | 推荐DPI | 说明 |
|------|--------|------|
| 普通文档 | 96-150 | 屏幕显示，基础打印 |
| 商业报告 | 150-200 | 清晰文字和图表 |
| 高清图表 | 200-300 | 精细数据可视化 |
