# Industry Research - 行业研究商业分析插件

Claude Code 插件，提供专业的行业研究与商业分析咨询服务。

## 功能特性

- 🌍 **全球市场分析**：全球市场与重点区域市场深度分析
- 📈 **行业热点趋势**：行业热点追踪与趋势预测
- 🏆 **竞争格局分析**：行业竞争格局与头部企业分析
- 👥 **消费者洞察**：用户画像、消费行为、需求洞察
- 📚 **品牌案例拆解**：成功品牌案例深度拆解与学习
- 💡 **行业洞见**：专业咨询视角的行业结论与建议

## 安装

```bash
# 复制到项目 .claude 目录
cp -r industry-research /path/to/your-project/.claude/plugins/
```

## 使用方法

### 综合行业分析

```
/industry-analyze <行业名称>
```

示例：
```
/industry-analyze 新能源汽车
/industry-analyze 预制菜
/industry-analyze AIGC
```

### 全球市场分析

```
/market-analysis <行业/产品> [区域]
```

示例：
```
/market-analysis 电动汽车 全球
/market-analysis 东南亚 电商
```

### 竞争格局分析

```
/competitor-analysis <行业>
```

### 消费者洞察

```
/consumer-insight <行业/产品>
```

### 品牌案例拆解

```
/brand-case <品牌名>
```

## 分析维度

| 维度 | 内容 | 信息源 |
|------|------|-------|
| 行业概况 | 市场规模、增长率、发展阶段 | 行业报告、统计局 |
| 全球市场 | 主要区域市场、进出口、供应链 | 海关数据、国际机构 |
| 行业热点 | 政策动态、技术突破、资本动向 | 财经媒体、政策文件 |
| 重点产品 | 产品矩阵、创新方向、技术路线 | 企业官网、电商平台 |
| 竞争格局 | 市场份额、头部企业、竞争态势 | 行业报告、财报 |
| 消费者 | 用户画像、消费动机、决策路径 | 调研报告、社交数据 |
| 品牌案例 | 成功路径、关键决策、可复制点 | 企业访谈、深度报道 |

## 咨询方法论

### 分析框架

```
1. 宏观环境 (PESTEL)
   ├── 政策环境 (Political)
   ├── 经济环境 (Economic)
   ├── 社会环境 (Social)
   ├── 技术环境 (Technological)
   ├── 环境因素 (Environmental)
   └── 法律因素 (Legal)

2. 市场规模分析
   ├── TAM (总可触达市场)
   ├── SAM (可服务市场)
   └── SOM (可获得市场)

3. 竞争分析
   ├── 波特五力模型
   ├── SWOT分析
   └── 竞争对手矩阵

4. 消费者研究
   ├── 用户画像
   ├── 消费旅程
   └── JTBD (待完成任务)
```

## 输出报告

分析报告保存到：`docs/industry-analysis/{行业名}-研究报告-{日期}.md`

报告包含：
- 行业概况与规模
- 全球市场与区域分析
- 行业热点与趋势
- 重点产品与技术
- 竞争格局分析
- 消费者洞察
- 品牌案例借鉴
- 行业结论与建议
