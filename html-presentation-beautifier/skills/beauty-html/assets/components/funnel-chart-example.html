漏斗图示例 - Funnel Chart Examples

McKinsey 风格转化流程可视化，横向条形图宽度递减实现漏斗效果。适用于销售漏斗、用户转化、网站分析等场景。

电商销售漏斗

关键洞察

总体转化率： 3.6%

从访问到购买的完整转化路径

  • 访问到加购转化率：30%
  • 加购到下单转化率：50%
  • 下单到支付转化率：24%
  • 需优化支付环节流失问题

技术实现 - Technical Implementation

核心实现逻辑

使用 Chart.js 横向条形图（bar），通过调整 indexAxis 和 barPercentage 实现宽度递减：

  • type: 'bar' - 使用条形图
  • indexAxis: 'y' - 横向显示
  • barPercentage 数组控制每个阶段的宽度
  • 使用 McKinsey 标准配色方案

设计规范 - Design Guidelines

  • 阶段数量：建议 3-5 个关键阶段
  • 宽度递减：从上到下宽度逐渐减小，形成漏斗形状
  • 转化率标注：每个阶段显示转化率或流失率
  • 颜色区分：每个阶段使用不同颜色，增强视觉层次
  • 数据标签：清晰显示每个阶段的数值和占比
