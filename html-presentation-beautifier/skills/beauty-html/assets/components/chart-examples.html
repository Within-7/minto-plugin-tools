McKinsey Style Chart Examples

Interactive chart examples using the McKinsey/BCG color palette

Bar Chart - Category Comparison

Use for comparing values across different categories. Bar charts are ideal for showing discrete comparisons between categories.

Line Chart - Trends Over Time

Use for showing trends and patterns over time periods. Line charts effectively visualize continuous data changes.

Pie Chart - Proportions

Use for showing percentage breakdowns of a whole. Best used when showing 5-7 categories maximum.

Multi-Metric Comparison

Use for comparing multiple metrics across categories. Grouped bar charts allow direct comparison of different measures.
