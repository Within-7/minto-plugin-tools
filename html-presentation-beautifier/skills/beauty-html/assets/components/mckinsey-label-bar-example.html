McKinsey标签图特点

  • 关键标注：在柱状图中直接标注关键洞察和结论
  • 数据驱动：基于数据得出明确的结论和行动项
  • 简洁表达：用最少的文字传达最重要的信息
  • 颜色编码：使用颜色区分正面/负面/中性趋势
  • 行动导向：标注直接关联下一步行动或决策
