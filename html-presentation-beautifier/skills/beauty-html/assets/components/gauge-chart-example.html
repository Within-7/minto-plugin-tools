技术实现 - Technical Implementation

核心实现逻辑

使用 Chart.js 环形图（doughnut）配置为半圆形：

  • type: 'doughnut' - 使用环形图
  • circumference: 180 - 半圆形状
  • rotation: 270 - 从左侧开始
  • cutout: '75%' - 中心孔径大小

动态颜色规则

  • ✅ 达成目标：绿色（#34c38f）
  • ⚠️ 接近目标：蓝色（#556EE6）
  • ❌ 未达标：橙红色（#F85d42）

设计规范 - Design Guidelines

  • 简洁设计：每个仪表盘只显示一个核心指标
  • 明确目标：始终显示目标值或基准线
  • 颜色语义：使用一致的颜色表示达成状态
  • 数值标注：在仪表盘下方清晰显示实际数值
  • 合理范围：设置最大值为目标的 1.1-1.2 倍
