## 🚀 快速开始

### 在图文形式选择阶段（步骤2.1）

```
步骤2.1: 图文形式选择

1. 读取示例索引 (INDEX.md)
   └─ 了解所有可用的示例类型

2. 分析当前页面特征
   ├─ 内容类型：概念性 vs 数据性
   ├─ 观点数量：1个、2-3个、4-6个、7+个
   ├─ 数据密度：无数据、少量数据、大量数据
   └─ 对比关系：并列、对比、流程、层级

3. 对比示例文档
   ├─ 读取所有匹配的示例文档
   ├─ 对比当前页面特征与示例特征
   └─ 计算匹配度

4. 选择最佳示例
   ├─ 选择匹配度最高的示例
   ├─ 如果有多个高匹配度示例，随机选择
   └─ 应用示例的布局和样式

5. 基于示例生成内容
   ├─ 使用示例的 HTML 结构
   ├─ 替换示例的内容为实际内容
   └─ 保持示例的样式和布局
```

## 📋 示例文档列表

### 1. 封面页示例 (01-cover-page.html)
- **适用**: 文档封面、章节封面
- **内容**: 标题、副标题、作者信息
- **布局**: 单列居中
- **颜色**: 主强调色 + 深蓝色渐变背景
- **特点**: 大标题56px黑色，副标题32px主强调色

### 2. 目录页示例 - 列表式 (08-table-of-contents.html)
- **适用**: 文档目录页、章节导航、内容概览
- **内容**: 3个或以上章节标题 + 描述 + 页码
- **布局**: 单列列表，章节编号左侧
- **颜色**: 主强调色章节编号 + 灰色描述
- **特点**: 清晰层级、悬停效果、章节编号32px加粗
- **设计**: 卡片式目录项，左边框悬停强调

### 3. 双列对比示例 (02-two-column-comparison.html)
- **适用**: 2个对象对比、A vs B
- **内容**: 左右两列，对比内容
- **布局**: 双列50:50，间距35px
- **颜色**: 主强调色 (#F85d42) vs 深蓝色 (#556EE6)
- **特点**: 对比强烈，对称布局

### 4. 三列并列示例 (03-three-column.html)
- **适用**: 3个并列观点、3个选项
- **内容**: 三列，并列内容
- **布局**: 三列33:33:33，间距35px
- **颜色**: 主强调色、深蓝色、蓝色渐变
- **特点**: 横向并列，充分利用空间

### 5. 卡片网格示例 (04-card-grid.html)
- **适用**: 4-6个并列观点、多个特性
- **内容**: 多个卡片，网格排列
- **布局**: 自适应网格（2x2, 2x3, 3x2），间距25px
- **颜色**: 规范6色系统
- **特点**: 自动适配，视觉丰富

### 6. 图表+文本示例 (05-chart-text.html)
- **适用**: 数据可视化 + 关键洞察
- **内容**: 图表（左）+ 文本（右）
- **布局**: 左图右文或上图下文
- **图表**: Chart.js，强制 width:100%
- **特点**: 图表修复配置，数据可视化

### 7. 数字强调示例 (06-data-emphasis.html)
- **适用**: 关键数据展示、宏观数据、利润/销售额/GDP
- **内容**: 1-3个超大数字 + 简短说明
- **布局**: 三列横向或单列纵向
- **颜色**: 黑色页眉 + 主强调色数字
- **特点**: 数字80-120px，数据占80%+视觉重心
- **设计**: 超大数字粗体，黑色页眉对比强烈

### 8. 雷达图+卡片示例 (07-radar-card-layout.html)
- **适用**: 能力评估、多维度分析、SWOT分析延伸
- **内容**: 雷达图(左) + 信息卡片(右)
- **布局**: 左右分栏，左侧雷达图+总评，右侧3个信息卡片
- **图表**: ECharts雷达图，5-8个维度
- **颜色**: 主强调色雷达 + 绿/红/橙图标
- **特点**: 多维能力可视化 + 结构化文本分析
- **设计**: 现代卡片风格，渐变背景，清晰层级

### 9. 品牌介绍页示例 (09-brand-intro-page.html)

## 🔍 匹配规则

### 优先级1: 数据角色
- 数据是主角（占比80%+）→ 数字强调示例
- 数据是配角（占比50-70%）→ 图表+文本示例
- 无数据 → 概念性布局

### 优先级2: 观点数量
- 1个观点 → 封面页、单列布局
- 2-3个观点 → 双列/三列布局
- 4-6个观点 → 卡片网格布局
- 7+个观点 → 分页或分组展示

### 优先级3: 对比关系
- 并列关系 → 多列布局、卡片网格
- 对比关系 → 双列对比布局
- 流程关系 → 时间轴、流程图
- 层级关系 → 金字塔图、树状图
