# 「学童优选」品牌分析 - HTML 演示文稿

## 快速开始

### 查看演示文稿

1. 打开 `presentation_demo/` 目录
2. 在浏览器中打开 `presentation_output.html`

或者使用命令行：
```bash
open presentation_demo/presentation_output.html
```

## 演示文稿导航

- **上一页**：点击 "← Previous" 按钮
- **下一页**：点击 "Next →" 按钮
- **键盘导航**：
  - `←` 或 `→` 箭头键：切换幻灯片
  - `空格键`：下一页
  - `ESC`：退出全屏
- **全屏模式**：点击 "⛶ Fullscreen" 按钮

## 演示文稿内容

本演示文稿包含 7 张幻灯片：

1. **标题页**：展示报告标题和日期
2. **执行摘要**：关键发现和结论概览
3. **数据分析 1**：展示市场数据图表和洞察
4. **数据分析 2**：展示结论数据图表
5. **数据分析 3**：展示建议数据图表
6. **详细发现**：展示财务结果详情
7. **结论与建议**：总结所有结论和建议

## 技术规格

- **文件格式**：HTML5
- **样式**：McKinsey/BCG 风格 CSS
- **图表**：Chart.js 交互式柱状图
- **配色**：专业咨询风格配色方案
- **响应式**：支持不同屏幕尺寸

## 系统要求

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 网络连接（用于加载 Chart.js CDN）

## 文件说明

- `presentation_output.html`：主演示文稿文件
- `styles.css`：McKinsey 风格样式表
- `script.js`：导航和交互脚本

## 测试数据来源

原始数据基于「学童优选」品牌的市场分析，包括：
- 市场规模和用户分析
- 产品线表现数据
- 对标案例研究（School Supply Boxes, Melissa & Doug）
- 财务预测和战略建议
