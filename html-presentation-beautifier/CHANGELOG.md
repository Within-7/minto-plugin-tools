# Changelog

All notable changes to the HTML Presentation Beautifier plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-26

### Added
- Initial release of HTML Presentation Beautifier plugin
- McKinsey-style design system with professional color palette
- 50+ chart types with intelligent selection
- 6-phase AI-powered workflow for presentation generation
- Single-file HTML output with inline CSS/JavaScript
- Interactive navigation and keyboard shortcuts
- Responsive design for desktop, tablet, and mobile
- Content integrity verification (100% preservation guarantee)
- Automatic quality review agent
- Multi-format support (Markdown, JSON, Text, HTML)
- `/beauty` command for quick presentation generation
- 5 specialized agents for presentation processing
- Comprehensive chart and visualization examples
- Template system for consistent slide generation

### Features
- **Document Parsing**: Extract structure, data, and conclusions
- **Slide Planning**: AI-powered content structuring
- **Design Application**: McKinsey style enforcement
- **Content Visualization**: Intelligent chart selection
- **HTML Generation**: Single-file output with Chart.js
- **Quality Review**: Automated integrity and style checks

### Documentation
- Complete skill documentation with workflow guides
- Chart examples index with 30+ visualization types
- Design system reference (McKinsey/BCG style)
- Template usage guide
- Agent documentation for each specialized subagent
- Plugin installation and usage guide
