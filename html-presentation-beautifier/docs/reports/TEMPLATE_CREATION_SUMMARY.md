## 🔧 占位符系统

模板使用 `{{PLACEHOLDER}}` 格式的占位符，便于替换：

### 主要占位符列表

| 占位符 | 用途 | 位置 |
|--------|------|------|
| `{{PRESENTATION_TITLE}}` | 浏览器标签标题 | `<title>` |
| `{{MAIN_TITLE}}` | 灯片 |
| `{{SUBTITLE}}` | 副标题 | 标题幻灯片 |
| `{{SECTION_TITLE}}` | 章节标题 | 内容幻灯片 |
| `{{BULLET_POINT_N}}` | 列表项 | 列表幻灯片 |
| `{{CHART_TITLE}}` | 图表标题 | 图表幻灯片 |
| `{{CONCLUSION_N_TITLE}}` | 结论标题 | 结论卡片 |
| `{{CONCLUSION_N_TEXT}}` | 结论内容 | 结论卡片 |
| `{{LEFT/RIGHT_COLUMN_TITLE}}` | 栏标题 | 两栏布局 |
| `{{LEFT/RIGHT_POINT_N}}` | 栏内容 | 两栏布局 |

### 使用示例

```javascript
// 读取模板
let html = readFile('presentatemplate.html');

// 替换占位符
html = html.replace('{{PRESENTATION_TITLE}}', '简优战略规划');
html = html.replace('{{MAIN_TITLE}}', '简优战略方向梳理');
html = html.replace('{{SUBTITLE}}', '聚焦「学童优选」品牌定位');

// 写入新文件
writeFile('output.html', html);
```
