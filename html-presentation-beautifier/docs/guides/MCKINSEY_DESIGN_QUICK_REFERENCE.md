# McKinsey 设计规范快速参考

## ⚠️ CRITICAL - 必须严格遵守

### 颜色规范（精确匹配）

```css
/* McKinsey 标准调色板 - 必须使用精确的 hex 值 */
:root {
  --primary-background: #FFFFFF;     /* 幻灯片背景 */
  --header-background: #000000;      /* 标题栏 */
  --primary-accent: #F85d42;         /* 主强调色 */
  --secondary-accent: #74788d;       /* 辅助文本 */
  --deep-blue: #556EE6;              /* 深蓝色 */
  --green: #34c38f;                  /* 绿色 */
  --blue: #50a5f1;                   /* 蓝色 */
  --yellow: #f1b44c;                 /* 黄色 */
  --text-dark: #333333;              /* 正文文本 */
  --text-black: #000000;             /* 标题文本 */
}
```

**使用规则**：
- ✅ 使用精确的 hex 值（包括大小写）
- ❌ 不使用近似值（如 `#F85D43` 代替 `#F85d42`）
- ❌ 不使用简写（如 `#FFF` 代替 `#FFFFFF`）
- ❌ 不使用非标准颜色

### 字体规范（精确范围）

| 元素类型 | 大小范围 | 字重 | 颜色 |
|---------|---------|------|------|
| 标题 | 48-64px | bold (700) | #000000 |
| 副标题 | 28-36px | bold (700) | #F85d42 |
| 正文 | 16-20px | normal (400) | #333333 |
| 图表标签 | 12-14px | normal | #333333 |
| 图表标题 | 28-36px | bold | #000000 |

**使用规则**：
- ✅ 标题必须在 48-64px 范围内
- ✅ 副标题必须在 28-36px 范围内
- ✅ 正文必须在 16-20px 范围内
- ❌ 不允许超出范围
- ❌ 不允许使用错误的字重

### 布局规范（精确数值）

| 元素 | 数值 | 说明 |
|------|------|------|
| 页边距 | 40-60px | 所有幻灯片 |
| 元素间距 | 20-30px | 内容块之间 |
| box-sizing | border-box | 必须 |
| 响应式断点 | 1200px, 768px | 桌面、平板、移动 |

**使用规则**：
- ✅ 页边距必须在 40-60px 范围内
- ✅ 元素间距必须在 20-30px 范围内
- ✅ 必须使用 `box-sizing: border-box`
- ❌ 不允许超出范围
- ❌ 不允许使用 `content-box`

## 🚨 常见错误

### 错误 1：颜色近似值

❌ **错误**：
```css
--primary-accent: #F85D43;  /* 小写错误 */
--header-background: #111;  /* 应为纯黑 */
```

✅ **正确**：
```css
--primary-accent: #F85d42;  /* 精确值 */
--header-background: #000000;  /* 纯黑 */
```

### 错误 2：字体大小超出范围

❌ **错误**：
```css
.title { font-size: 40px; }  /* 太小，应≥48px */
.subtitle { font-size: 40px; }  /* 太大，应≤36px */
```

✅ **正确**：
```css
.title { font-size: 56px; }  /* 48-64px ✓ */
.subtitle { font-size: 32px; }  /* 28-36px ✓ */
```

### 错误 3：内容精简

❌ **错误**：
```markdown
源文档：
- 要点 1
- 要点 2
- 要点 3
- 要点 4
- 要点 5

HTML（错误）：
- 要点 1
- 要点 2
- 要点 3 等  <!-- 使用"等"，CRITICAL -->
```

✅ **正确**：
```markdown
HTML（正确）：
- 要点 1
- 要点 2
- 要点 3
- 要点 4
- 要点 5  <!-- 完整保留 -->
```

### 错误 4：数据精度损失

❌ **错误**：
```markdown
源文档：1723.498 亿美元
HTML（错误）：1723 亿美元  <!-- CRITICAL -->
```

✅ **正确**：
```markdown
HTML（正确）：1723.498 亿美元  <!-- 精度一致 -->
```

### 错误 5：图表使用非标准颜色

❌ **错误**：
```javascript
backgroundColor: ['#FF0000', '#00FF00']  /* 非标准颜色 */
```

✅ **正确**：
```javascript
backgroundColor: ['#F85d42', '#34c38f']  /* McKinsey 颜色 */
```
